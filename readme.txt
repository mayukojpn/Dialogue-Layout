== Description ==

An easy way to make conversation layout with avatars in post contents.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/dialogue-layout` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Insert [dialogue] shortcode through post edit page.
1. With installing [Shortcake](https://wordpress.org/plugins/shortcode-ui/) plugin you can use the shortcode easily.
